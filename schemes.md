	`ngie siz pieng ieo jia, ngie suah pieng ieo hua.`

	`ieo traq ie tsuaq diaq traq diaq, cyo saq huae reng.`

### 模糊音

全拼方案內置模糊音，以提供更舒適的輸入體驗：

- 聲母 `dr tr sr` 可用 `zh ch sh` 代替；
- 韻母 `iz uz` 可用 `i u` 代替；
- 韻母 `eo` 可用 `ao` 代替，位於 `b p m` 後也可用 `o` 代替；
- 韻母 `ae uae` 可用 `ai uai` 代替；
- 韻母 `uei iou` 可用 `ui iu` 代替；
- 韻尾 `n ng` 可混用；
- 韻母 `iaq/uaq/yaq` 可用 `iq/uq/yq` 代替；
- 入聲韻尾 `q` 可用 `h` 代替；

### 拼寫

**需要注意的是**，儘管如同漢語拼音下的雙拼方案一樣，零聲母分爲 `o/w/y` 三個，但在本方案中，零聲母音節被嚴格拆分爲 **零聲母** + **韻母**，輸入時要鍵入完整韻母的對應按鍵。

例如：

在 小鶴雙拼 中，`uan` 音節被拆分爲 `w` + `an`，即鍵入 **W** + **J**；

但在 本方案 中，`uan` 音節被拆分爲 `合口零聲母` + `uan`，即鍵入 **W** + **R**。

## 並擊方案

以並擊方式輸入平遙話城關音。兩種拼音方案下的並擊規則相同，區別僅在於顯示方案。以下以晉拼方案爲例介紹。參考 [宮保拼音](https://github.com/rime/rime-combo) 創作。

### 按鍵佈局

    --------------------------------------
    指法   左4 左3 左2 左2  右2 右2 右3 右4
    --------------------------------------
    上排   ts   l  d   t       u  -i   o
    中排    s   h  g*  k       i*  q   e
    下排   ds   r  b   p   er  y  -u   o'
    拇指                   a*
    --------------------------------------
    指法                  右1
    --------------------------------------
    
標「*」的按鍵爲拇指、食指的基準鍵位，可幫助定位其他按鍵。

o' 是 o 的活用鍵位。
    
### 組合聲母

       r+b = ⟨m⟩,   l+d = ⟨n⟩,     h+g = ⟨r⟩,
      ds+r = ⟨dr⟩, ts+l = ⟨tr⟩,    s+h = ⟨sr⟩,
    ds+r+b = ⟨z⟩, s+h+g = ⟨nr⟩, ts+l+d = ⟨nz⟩,

### 組合韻母及活用

括弧內爲活用組合

      a-u = ⟨eo⟩,   i+a-u = ⟨ieo⟩,
      o-i = ⟨ei⟩,     u-i = ⟨uei⟩,
      a-i = ⟨ae⟩,   u+a-i = ⟨uae⟩,
     o'-u = ⟨ou⟩,     y-u = ⟨iou⟩,
    a+q+e = ⟨an⟩, a+i+q+e = ⟨ian⟩,
       a+i+q+e (a+u-i+o) = ⟨uan⟩,
      q+e = ⟨eng⟩,  i+q+e = ⟨ieng⟩,  
           u+q+e (u-i+o) = ⟨ueng⟩,
         y+q+e  (y-u+o') = ⟨yeng⟩
